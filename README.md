# InsureX
InsureX is a full-stack Insurance Management System designed to digitalize and streamline the management of insurance policies, customers, premium payments, and claims. The application provides a centralized platform for administrators, agents, and customers to interact with insurance data securely and efficiently.

The backend is built using Java and Spring Boot, following RESTful architecture principles, with MySQL as the relational database for persistent storage. The frontend is developed using React and Tailwind CSS to deliver a modern, responsive, and user-friendly interface.

Technology Stack :
Backend : Java, Spring Boot, Spring Security & JWT, Hibernate / JPA, MySQL
Frontend : React, Tailwind CSS
